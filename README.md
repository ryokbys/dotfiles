# dotfiles
RK's personal dotfiles.

## Prerequisite

- zsh
  - oh-my-zsh
- emacs
  - prelude

## Install

Download the dotfiles project directory.
```shell
git clone https://github.com/ryokbys/dotfiles.git dotfiles
```

Install the dotfiles using `bin/install.sh`
```shell
cd dotfiles
./bin/install.sh
```
